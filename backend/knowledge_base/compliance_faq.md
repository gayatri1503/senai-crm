# Compliance FAQ

## HIPAA Compliance
- We offer a Business Associate Agreement (BAA) for customers in healthcare
- BAA is available on Pro and Enterprise plans only
- Data is encrypted at rest (AES-256) and in transit (TLS 1.2+)
- Servers are hosted in AWS us-east-1 and eu-west-1 regions
- HIPAA-compliant configuration requires enabling additional data controls in settings

## GDPR Compliance
- We are fully GDPR compliant as a data processor
- Data Processing Agreement (DPA) available upon request
- Data portability requests (Article 20) must be fulfilled within 30 days
- Data erasure requests (Article 17) processed within 72 hours
- EU customer data stored in eu-west-1 (Ireland) by default
- Sub-processors list available at ourplatform.com/sub-processors

## SOC 2 Type II
- We hold SOC 2 Type II certification (renewed annually)
- Latest audit report available under NDA to Enterprise customers
- Controls cover: security, availability, processing integrity, confidentiality

## Data Residency
- US customers: data stored in AWS us-east-1 (Virginia)
- EU customers: data stored in AWS eu-west-1 (Ireland)
- Custom data residency available for Enterprise customers (additional cost)

## ISO 27001
- ISO 27001 certification in progress (expected Q2 2024)
- Current security controls align with ISO 27001 framework
- Penetration testing conducted annually by third-party firm
