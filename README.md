# SenAI CRM — Agentic CRM Intelligence Platform

AI-powered CRM system with autonomous email triage, RAG pipeline, and real-time business intelligence.

## Stack
- **Backend**: FastAPI + PostgreSQL + ChromaDB
- **LLM**: Groq (Llama 3.1)
- **Embeddings**: sentence-transformers
- **Frontend**: React + Vite
- **Agent pattern**: ReAct (Reason + Act)
